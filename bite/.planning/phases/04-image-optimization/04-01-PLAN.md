    2. Create `app/Services/ImageService.php` with these methods:

    ```php
    class ImageService
    {
        public function supportsWebp(): bool
        // Returns true if gd_info()['WebP Support'] is truthy. Cache result in a static property.

        public function processUpload(string $storedPath, string $disk = 'public'): string
        // $storedPath is the relative path returned by $file->store() e.g. "products/abc123.jpg"
        // Steps:
        //   a) Get absolute path via Storage::disk($disk)->path($storedPath)
        //   b) Determine output extension: 'webp' if supportsWebp(), else 'jpg'
        //   c) Derive base name without extension: e.g. "products/abc123"
        //   d) Create Intervention ImageManager with GD driver
        //   e) For each variant ['thumb' => 200, 'card' => 400, 'full' => 800]:
        //      - Read original image
        //      - Scale down (longest edge) — use $image->scaleDown(width: $size, height: $size)
        //      - Encode: toWebp(quality: 80) or toJpeg(quality: 85) based on supportsWebp()
        //      - Save to: "{baseName}-{variant}.{ext}" on same disk
        //   f) Delete original file: Storage::disk($disk)->delete($storedPath)
        //   g) Return the full-size variant path: "{baseName}-full.{ext}"
        //      This becomes the new image_url value stored on the Product model.

        public function deleteVariants(string $imageUrl, string $disk = 'public'): void
        // $imageUrl is the stored image_url e.g. "products/abc123-full.webp"
        // Derive baseName by removing the "-full" suffix and extension
        // Delete: {baseName}-thumb.{ext}, {baseName}-card.{ext}, {baseName}-full.{ext}
        // Use Storage::disk($disk)->delete() — no exception on missing files
    ```

    3. Create `app/Helpers/image.php` with a global function:

    ```php
    function productImage(?App\Models\Product $product, string $variant = 'card'): ?string
    // If $product is null or $product->image_url is null, return null
    // $product->image_url is e.g. "products/abc123-full.webp"
    // Replace "-full." with "-{$variant}." in the path
    // Return "/storage/{result}"
    // Valid variants: 'thumb', 'card', 'full'
    ```

    4. Register the helper in composer.json autoload.files array, alongside the existing currency.php entry:

    ```json
    ]
    ```

For the GD WebP disabled test, create a subclass or use a mock that overrides `supportsWebp()` to return false.

    Image quality settings: WebP at quality 80 (good balance for food photos), JPEG fallback at quality 85.
