**CRITICAL: Product shop_id is guarded.** The DemoMenuSeeder uses `Product::create` with shop_id in the array, but this silently ignores shop_id because it's in $guarded. Use `Product::forceCreate()` instead to set shop_id, OR set it explicitly after creation. The safest pattern:

```php
$product = new Product($data);
$product->shop_id = $shop->id;
$product->category_id = $category->id;
$product->save();
```

    'currency_symbol' => 'ر.ع.',
    'currency_decimals' => 3,
    'tax_rate' => 0,
