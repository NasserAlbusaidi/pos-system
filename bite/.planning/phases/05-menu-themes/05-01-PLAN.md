**Step 1: Download font files.**

Download WOFF2 font files from Google WebFonts Helper (gwfh.mranftl.com) and place in `public/fonts/`:

- Inter Regular (400): download latin subset WOFF2 from `https://gwfh.mranftl.com/fonts/inter?subsets=latin` — save as `public/fonts/Inter-Regular.woff2`
- Inter SemiBold (600): same source, weight 600 — save as `public/fonts/Inter-SemiBold.woff2`
- DM Sans Regular (400): download from `https://gwfh.mranftl.com/fonts/dm-sans?subsets=latin` — save as `public/fonts/DMSans-Regular.woff2`
- DM Sans Medium (500): same source, weight 500 — save as `public/fonts/DMSans-Medium.woff2`
- DM Serif Display Regular (400): download from `https://gwfh.mranftl.com/fonts/dm-serif-display?subsets=latin` — save as `public/fonts/DMSerifDisplay-Regular.woff2`

Use `curl` to download from the gwfh.mranftl.com API. The API format is:
```
https://gwfh.mranftl.com/api/fonts/{font-id}?download=zip&subsets=latin&formats=woff2&variants={variant}
```

If gwfh.mranftl.com download fails, use the direct Google Fonts CSS API approach:
1. `curl -s -H "User-Agent: Mozilla/5.0" "https://fonts.googleapis.com/css2?family=Inter:wght@400;600&display=swap"` — extract woff2 URL from `src:` in response
2. Download each woff2 URL and save to `public/fonts/`
3. Repeat for DM Sans (wght@400;500) and DM Serif Display (wght@400)
