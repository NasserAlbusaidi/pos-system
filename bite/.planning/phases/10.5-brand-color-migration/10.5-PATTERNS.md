## Shared Patterns

### Hex → RGB Triple Helper
**Source:** `resources/views/layouts/app.blade.php` lines 33–42
**Apply to:** any future Blade-level inline-style branding partial; in 10.5 used by the scoped preview in `shop-settings.blade.php`.

```php
$toRgb = function ($hex) {
    $hex = ltrim((string) $hex, '#');
    if (strlen($hex) === 3) {
        $hex = $hex[0] . $hex[0] . $hex[1] . $hex[1] . $hex[2] . $hex[2];
    }
    if (strlen($hex) !== 6) {
        return '0 0 0';
    }
    [$r, $g, $b] = sscanf($hex, '%02x%02x%02x');
    return "{$r} {$g} {$b}";
};
```

### Token Reference Convention
**Source:** `resources/css/app.css` throughout (e.g., line 217–223)
**Apply to:** every CSS rule using a token in 10.5.

```css
color: rgb(var(--ink));
background-color: rgb(var(--ink));
box-shadow: 0 8px 24px -14px rgb(10 15 24 / 0.9);
```

Use `rgb(var(--token))` (no alpha) or `rgb(var(--token) / 0.NN)` (with alpha). NEVER `var(--token)` directly inside a color property — the triple format requires the `rgb()` wrapper.
